# 🌟 Tribute Page – Dr. A.P.J. Abdul Kalam

This is a simple tribute page dedicated to the Missile Man of India, Dr. A.P.J. Abdul Kalam. It is built using basic HTML and CSS.

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📁 Folder Structure
- `TributePage.html` → Main HTML file 
- `TributePage.css` → Main stylesheet 
- `README.md` → Project documentation

## 🚀 How to Run

1. Download or clone the repository:  
   `git clone https://github.com/Prakruthi-G-H/TributePage.git`

2. Open the folder in any browser or use Live Server in VS Code  
3. Open `TributePage.html` to view the homepage  
4. No backend or server setup required
